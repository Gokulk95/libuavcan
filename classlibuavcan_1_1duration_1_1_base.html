Detailed Description

template<typename Type, typename USecT = libuavcan::DefaultMicrosecondSignedType>
class libuavcan::duration::Base< Type, USecT >

Protected base class for duration values. This provides a common implementation for various duration datatypes and enforces two concepts:

  1. duration math is saturing – MAX_DURATION + 1 == MAX_DURATION
  2. durations are signed integers – By default 8 byte integers but USecT can be redefined by a specialization.
Template Parameters
TypeThe type of the derived class. This must be an empty type. All storage will be provided by this base class.
USecTThe datatype returned when retrieving durations from realizations of this base class. This type must be signed.
