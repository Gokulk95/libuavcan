◆ BytePaddingPattern

constexpr std::uint8_t libuavcan::media::CAN::BytePaddingPattern = 0x55

Bit pattern to fill padding bytes with. The UAVCAN specification mandates this value but the actual value of padding bytes must be ignored when receiving messages.

When transmitting use this pattern to minimize the number of stuff bits added by the CAN hardware.
